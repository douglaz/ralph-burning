   - Current: [engine.rs](/root/new-ralph-burning/ralph-burning-rewrite/src/contexts/workflow_composition/engine.rs#L948) computes `will_retry` once from the current token state, but the next loop iteration immediately emits `stage_entered` and rewrites the running snapshot at [engine.rs](/root/new-ralph-burning/ralph-burning-rewrite/src/contexts/workflow_composition/engine.rs#L812) without a fresh cancellation check. Add regression coverage for cancellation arriving after the retry decision but before the next attempt starts.

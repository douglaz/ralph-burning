2. **Added CWD-relative fallback for compile-time paths** (`src/contexts/conformance_spec/catalog.rs`, `src/cli/conformance.rs`): The conformance runner uses `CARGO_MANIFEST_DIR` at compile time to locate the `tests/conformance/features/` directory and the `src/` directory for the cutover guard. When the binary is built by Nix, this path points to the build sandbox (`/build/ralph-burning-rewrite/...`) which doesn't exist at runtime. Both call sites now check whether the compile-time path exists and fall back to CWD-relative resolution (`tests/conformance/features` and `src` respectively).

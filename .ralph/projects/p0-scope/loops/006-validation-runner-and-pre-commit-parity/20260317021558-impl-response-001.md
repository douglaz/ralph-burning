Replaced `let _ = validation::persist_pre_commit_evidence(...)` at `engine.rs:3083` (pre-commit evidence persistence) with the same pattern. Both locations now enforce the failure invariant: if supporting payload/artifact persistence fails after command execution, the run does not advance, does not synthesize approval or remediation, and remains resumable from the pre-transition stage boundary.

4. **Conformance improvement** (Recommended Improvement 1): Added unit test `standard_flow_review_invocation_context_contains_local_validation` in `workflow_engine_test.rs` using the `RecordingAdapter` pattern.

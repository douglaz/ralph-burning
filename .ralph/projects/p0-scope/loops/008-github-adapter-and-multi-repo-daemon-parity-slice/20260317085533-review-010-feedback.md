Current: `sync_label_for_task()` removes every status label before adding the new one at [github_intake.rs](/root/new-ralph-burning/ralph-burning-rewrite/src/contexts/automation_runtime/github_intake.rs#L316) and [github_intake.rs](/root/new-ralph-burning/ralph-burning-rewrite/src/contexts/automation_runtime/github_intake.rs#L329). If that add fails, `process_task_multi_repo()` propagates the error after the task has already moved forward at [daemon_loop.rs](/root/new-ralph-burning/ralph-burning-rewrite/src/contexts/automation_runtime/daemon_loop.rs#L440), [daemon_loop.rs](/root/new-ralph-burning/ralph-burning-rewrite/src/contexts/automation_runtime/daemon_loop.rs#L502), [daemon_loop.rs](/root/new-ralph-burning/ralph-burning-rewrite/src/contexts/automation_runtime/daemon_loop.rs#L531), and [daemon_loop.rs](/root/new-ralph-burning/ralph-burning-rewrite/src/contexts/automation_runtime/daemon_loop.rs#L549). Later cycles only poll issues that still carry a status label at [github_intake.rs](/root/new-ralph-burning/ralph-burning-rewrite/src/contexts/automation_runtime/github_intake.rs#L115) and [github_intake.rs](/root/new-ralph-burning/ralph-burning-rewrite/src/contexts/automation_runtime/github_intake.rs#L160), and only execute `Pending` tasks at [daemon_loop.rs](/root/new-ralph-burning/ralph-burning-rewrite/src/contexts/automation_runtime/daemon_loop.rs#L327). `daemon reconcile` only repairs leases/worktrees, not GitHub labels, at [daemon.rs](/root/new-ralph-burning/ralph-burning-rewrite/src/cli/daemon.rs#L409). A transient label failure can therefore leave an issue with no status label and a non-pending task the daemon will no longer see or repair.
Expected: status-label updates need a repairable failure mode. Either make the status-label change atomic from the daemon’s perspective, or persist/surface a durable mismatch and have reconcile restore the correct status label before the task becomes invisible.
Reference: Slice 8 acceptance criteria for status-label mapping, partial label-write failures, and later reconcile/repair behavior.
